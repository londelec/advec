Overview

Advantech Linux EC Drivers currently include brightness, gpio, hwmon, watchdog, led, eeprom and common. All the ec related platform drivers depends on the mfd-ec driver.

Version Information

[v2.20] : 2022/10/11

  • Support Debian package to install on Debian-based OS, such as Ubuntu, Debian.
  • Support Linux DKMS on Debian-based OS
  • Support auto insert modules on Debian-based OS
  • Driver example complies automatically with DKMS

[v2.16] : 2022/08/10

  • Update EC Watchdog driver to 1.15 to be compatible with Ubuntu watchdog service.
  • Update EC mfd driver to 1.08 and EC GPIO driver to 1.11 to support PoE-enabled UNO-2484G.
  • Restore the working state of GPIO and PoE when resuming from S3, S5 power state

[v2.14] : 2021/07/14

  • Add support the platform UNO-137-E1,UNO-410-E1,WISE-5580.

[v2.11] : 2019/09/10

  • Add support 32-bit example program in 64-bit system.

[v2.09-Beta1] : 2018/05/11

  • Replace EC POWER driver to EC COMMON driver, support common-gpio and common-power.

Software Prerequisite

EAPI Library V1.0.14 or later.

OS Information

Ubuntu Desktop 20.04.3 LTS, kernel 5.11.0-27-generic

Notes

Drivers will be installed in the folder /usr/src/advantech/advec-{package version} and loaded during the booting process. When the system boots up, all of these drivers will be loaded. Each driver is provided with test program which can be found in /usr/src/advantech/advec-{package version}/example/.

Important Drivers Version
